# Fonti Dati Istituzionali Italiane per l'Analisi Sanitaria

Questo documento elenca tutte le fonti dati istituzionali italiane utilizzate e disponibili per l'analisi delle patologie a elevata complessità assistenziale.

---

## Fonti Dati Online (Portali Interattivi)

### ISTAT - Esplora Dati Sanità
- **URL**: https://esploradati.istat.it/databrowser/#/it/dw/categories/IT1,Z0810HEA,1.0/HEA_SERVICES/
- **Contenuto**: Dati interattivi su servizi sanitari, visite specialistiche, ricoveri
- **Tipo**: Portale interattivo con export CSV/Excel

### ISTAT - Health for All Italia
- **URL**: https://www.istat.it/sistema-informativo-6/health-for-all-italia/
- **Contenuto**: 4.000 indicatori sanitari su patologie croniche, mortalità, assistenza
- **Tipo**: Software desktop (richiede download)

### AGENAS - Portale Statistico
- **URL Specialistica Ambulatoriale**: https://stat.agenas.it/web/index.php?r=public%2Findex&report=12
- **URL Liste d'Attesa**: https://stat.agenas.it/web/index.php?r=public%2Findex&report=40
- **Contenuto**: Volumi prestazioni specialistiche, tempi di attesa per regione
- **Tipo**: Dashboard Power BI interattiva

### AGENAS - Sclerosi Multipla (GLASM)
- **URL**: https://www.agenas.gov.it/aree-tematiche/organizzazione-dei-servizi-sanitari/sclerosi-multipla-glasm
- **Contenuto**: Gruppo di Lavoro Assistenza Sclerosi Multipla
- **Tipo**: Documentazione e linee guida

### Ministero della Salute - Open Data
- **URL**: https://www.dati.salute.gov.it/
- **Contenuto**: Dataset aperti su farmacie, dispositivi medici, SDO, personale SSN
- **Licenza**: Italian Open Data License v2.0

### Dati.gov.it - Sezione Salute
- **URL**: https://www.dati.gov.it/view-dataset?groups=salute
- **Contenuto**: Aggregatore nazionale open data sanitari
- **Tipo**: Catalogo con link a dataset regionali e nazionali

### SISTAN - Sistema Statistico Nazionale
- **URL**: https://www.sistan.it/index.php?id=688
- **Contenuto**: Programma Statistico Nazionale - Area Sanità
- **Tipo**: Documentazione metodologica e pianificazione statistica

### ISS - EpiCentro PASSI e PASSI d'Argento
- **URL PASSI**: https://www.epicentro.iss.it/passi/
- **URL PASSI d'Argento**: https://www.epicentro.iss.it/passi-argento/dati/croniche
- **Contenuto**: Sorveglianza patologie croniche, fattori di rischio, stili di vita
- **Tipo**: Report e dati regionali

## Note sull'Aggiornamento

I dataset istituzionali vengono generalmente aggiornati con le seguenti frequenze:

| Fonte | Frequenza Aggiornamento |
|-------|------------------------|
| Rapporto SDO | Annuale (aprile) |
| Open Data Ministero Salute | Annuale/Semestrale |
| ISTAT HFA | Annuale |
| PASSI/PASSI d'Argento | Continuo (pool biennali) |
| Orphadata | Trimestrale |
| Rapporto Osservatorio Salute | Annuale (dicembre) |

Per mantenere i dati aggiornati, si consiglia di verificare periodicamente le fonti originali.
