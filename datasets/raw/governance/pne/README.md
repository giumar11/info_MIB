# PNE - Programma Nazionale Esiti

**Fonte:** AGENAS
**URL:** https://pne.agenas.it/
**Descrizione:** Indicatori di esito e performance delle strutture ospedaliere italiane
**Aggiornamento:** Annuale
**Formato:** Dashboard interattiva + Report PDF

## Come accedere ai dati
I dati PNE sono disponibili tramite dashboard interattiva. Per il download:
1. Accedere a https://pne.agenas.it/
2. Selezionare indicatori e regione
3. Esportare in formato Excel/CSV
