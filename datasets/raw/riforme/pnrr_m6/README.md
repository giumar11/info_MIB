# PNRR Missione 6 - Salute

**Fonte:** Governo Italiano / Italia Domani
**URL:** https://italiadomani.gov.it/it/il-piano/missioni-pnrr/salute.html
**Budget:** 15,63 miliardi di euro
**Descrizione:** Investimenti per la riforma del sistema sanitario italiano

## Componenti
### M6C1 - Reti di prossimità, strutture e telemedicina (7 mld)
- 1.350 Case della Comunità
- 400 Ospedali di Comunità
- 600 Centrali Operative Territoriali
- Telemedicina

### M6C2 - Innovazione, ricerca e digitalizzazione (8,63 mld)
- Ammodernamento parco tecnologico
- Fascicolo Sanitario Elettronico
- Ricerca biomedica
