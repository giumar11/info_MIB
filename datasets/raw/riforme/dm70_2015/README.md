# DM 70/2015 - Standard Qualitativi Rete Ospedaliera

**Fonte:** Ministero della Salute / Gazzetta Ufficiale
**URL:** https://www.gazzettaufficiale.it/eli/id/2015/06/04/15A04227/sg
**Data:** 2 Aprile 2015
**Descrizione:** Regolamento recante definizione degli standard qualitativi, strutturali, tecnologici e quantitativi relativi all'assistenza ospedaliera

## Standard principali
- **Posti letto**: 3,7 per 1.000 abitanti (di cui 0,7 riabilitazione/lungodegenza)
- **Tasso di ospedalizzazione**: 160 per 1.000 abitanti
- **Volumi minimi**: Soglie per procedure chirurgiche complesse

## Classificazione ospedali
- DEA I livello
- DEA II livello
- Presidi ospedalieri di base
- Ospedali di zona disagiata
