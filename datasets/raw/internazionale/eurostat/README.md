# Eurostat - System of Health Accounts (SHA 2011)

**Fonte:** Eurostat (Ufficio Statistico UE)
**URL:** https://ec.europa.eu/eurostat/web/health/data/database
**Descrizione:** Statistiche sanitarie armonizzate per i paesi UE secondo il framework SHA 2011
**Aggiornamento:** Annuale
**Formato:** Database online + Export CSV/Excel

## Classificazioni SHA 2011
- **HC**: Funzioni di cura (curativa, riabilitativa, long-term, etc.)
- **HP**: Fornitori di servizi (ospedali, ambulatori, farmacie, etc.)
- **HF**: Schemi di finanziamento (pubblico, privato, out-of-pocket)

## Dataset chiave
- hlth_sha11_hc: Spesa per funzione
- hlth_sha11_hp: Spesa per fornitore
- hlth_sha11_hf: Spesa per schema di finanziamento
