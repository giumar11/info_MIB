# OpenBDAP - Finanza degli Enti del SSN

**Fonte:** Ragioneria Generale dello Stato (RGS) / MEF
**URL:** https://openbdap.rgs.mef.gov.it/
**Descrizione:** Dati di bilancio e spesa degli enti del Servizio Sanitario Nazionale
**Aggiornamento:** Trimestrale/Annuale
**Formato:** Dashboard interattiva + Export CSV/Excel

## Aree tematiche
- Conto economico consolidato
- Spesa per funzione
- Spesa per natura economica
- Confronti regionali
