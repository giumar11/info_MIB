# PDTA - Regione Friuli Venezia Giulia

## Quadro generale

Il FVG ha avviato la Rete Oncologica Regionale nel febbraio 2025, con il Piano 2025-2027.
PDTA regionali in fase di sviluppo (completamento entro 30/06/2026).
