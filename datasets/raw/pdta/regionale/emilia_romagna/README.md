# PDTA - Regione Emilia-Romagna

## Quadro generale

L'Emilia-Romagna ha un approccio sistematico alla costruzione dei PDTA con una guida
metodologica regionale prodotta dall'ASSR (2013). Rete Oncologica ed Emato-Oncologica
regionale (DGR 2316/2022).

## Delibere principali

| Delibera | Oggetto | Anno |
