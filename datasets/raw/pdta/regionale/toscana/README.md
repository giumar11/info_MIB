# PDTA - Regione Toscana

## Quadro generale

La Toscana ha un sistema PDTA molto sviluppato con forte coinvolgimento dell'ARS
(Agenzia Regionale di Sanita). 3 ASL (Centro, Sud-Est, Nord-Ovest).
Rete Toscana Malattie Rare molto attiva.
